**Professional-grade portfolio risk management platform for investors, advisors, and financial firms**

**Live Platform:** [frmquant.com] (https://www.frmquant.com)

- **AI-Powered Intelligence** - Built using Claude API for superior language quality and numerical reasoning

- **Real-Time Risk Monitoring** - Live portfolio health scoring with natural language insights

- **Frontend:** React, TypeScript
- **Backend:** Node.js, Firebase
- **Cloud:** Google Cloud Platform
- **AI:** Claude API (primary LLM for financial analysis)

**Status:** Live in production with active paying customers
